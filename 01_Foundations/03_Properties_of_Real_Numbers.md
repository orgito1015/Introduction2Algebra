## 📖 Key Concepts

| Property | Addition | Multiplication |
|----------|----------|----------------|
| **Commutative** | `a + b = b + a` | `ab = ba` |
| **Associative** | `(a + b) + c = a + (b + c)` | `(ab)c = a(bc)` |
| **Distributive** | `a(b + c) = ab + ac` | — |
| **Identity** | `a + 0 = a` | `a × 1 = a` |
| **Inverse** | `a + (-a) = 0` | `a × (1/a) = 1` (a ≠ 0) |

## ✏️ Worked Examples

### Example 1 — Distributive Property

**Problem:** Expand `4(x + 3)`.

**Solution:** `4(x + 3) = 4x + 12`

---

### Example 2 — Commutative Property

**Problem:** Simplify `5 + x + 3`.

**Solution:** By the commutative and associative properties: `5 + 3 + x = 8 + x`

## 🧪 Practice Problems

1. Use the distributive property: `3(2x - 5)`.
2. Identify the property: `(2 + 7) + 1 = 2 + (7 + 1)`.
3. Find the additive inverse of `−9`.

2. Associative property of addition
3. `9`
