## 📖 Key Concepts

- **Expression** — a combination of numbers, variables, and operations (no equals sign).
- **Like terms** — terms with the same variable(s) raised to the same power (e.g., `3x` and `7x`).
- **Simplifying** — combining like terms and applying properties to write an equivalent, shorter expression.
- **Evaluating** — substituting a value for each variable and computing the result.

## ✏️ Worked Examples

### Example 1 — Simplify

**Problem:** Simplify `4x + 3 - x + 7`.

**Solution:** Combine like terms:  
`(4x - x) + (3 + 7) = 3x + 10`

---

### Example 2 — Evaluate

**Problem:** Evaluate `2x² - 3x + 1` when `x = 2`.

**Solution:**  
`2(2)² - 3(2) + 1 = 2(4) - 6 + 1 = 8 - 6 + 1 = 3`

## 🧪 Practice Problems

1. Simplify: `5y - 2y + 8 - 3`.
2. Evaluate `3a + 2b` when `a = 4`, `b = -1`.
3. Simplify: `2(x + 4) - 3x`.

<details>
<summary>Answers</summary>

1. `3y + 5`
2. `3(4) + 2(-1) = 12 - 2 = 10`
3. `2x + 8 - 3x = -x + 8`

</details>
