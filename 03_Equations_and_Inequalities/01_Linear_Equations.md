## 📖 Key Concepts

- **Equation** — a mathematical statement that two expressions are equal (uses `=`).
- **Linear equation** — an equation where the highest power of the variable is `1`.
- **Solution** — the value(s) of the variable that make the equation true.

### Standard Forms

| Form | Equation | Notes |
|------|----------|-------|
| Standard | `ax + b = c` | Basic one-variable form |
| Slope-intercept | `y = mx + b` | m = slope, b = y-intercept |
| Standard (two-variable) | `ax + by = c` | Useful for graphing |

## ✏️ Worked Examples

### Example 1

**Problem:** Is `x = 3` a solution to `2x + 1 = 7`?

**Solution:** Substitute: `2(3) + 1 = 6 + 1 = 7` ✅ Yes, `x = 3` is a solution.

## 🧪 Practice Problems

1. Verify that `x = -2` is a solution to `3x + 4 = -2`.
2. Write the equation "5 more than twice a number equals 13" in algebraic form.
3. Identify the slope and y-intercept of `y = -2x + 5`.

<details>
<summary>Answers</summary>

1. `3(-2) + 4 = -6 + 4 = -2` ✅
2. `2n + 5 = 13`
3. Slope: `-2`, y-intercept: `5`

</details>
