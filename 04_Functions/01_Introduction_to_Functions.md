## 📖 Key Concepts

- **Function** — a rule that assigns exactly **one output** to each input.
- **Domain** — the set of all valid input values (`x`).
- **Range** — the set of all possible output values (`y` or `f(x)`).
- **Function notation** — `f(x)` reads as "f of x"; `f(3)` means "evaluate f at x = 3".
- **Vertical Line Test** — a graph represents a function if every vertical line crosses it at most once.

## ✏️ Worked Examples

### Example 1

**Problem:** Given `f(x) = 2x + 1`, find `f(4)`.

**Solution:** `f(4) = 2(4) + 1 = 9`

---

### Example 2

**Problem:** Is the relation `{(1,2), (2,3), (1,4)}` a function?

**Solution:** No — the input `1` maps to two different outputs (`2` and `4`).

## 🧪 Practice Problems

1. Given `g(x) = x² - 3`, find `g(0)` and `g(-2)`.
2. Determine the domain of `f(x) = 1 / (x - 5)`.
3. Does `y² = x` define y as a function of x? Explain.

<details>
<summary>Answers</summary>

1. `g(0) = -3`, `g(-2) = 4 - 3 = 1`
2. Domain: all real numbers except `x = 5`, i.e., `(-∞, 5) ∪ (5, ∞)`
3. No — for `x = 4`, both `y = 2` and `y = -2` are solutions.

</details>
