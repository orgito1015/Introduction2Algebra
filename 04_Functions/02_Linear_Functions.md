## 📖 Key Concepts

- **Linear function** — a function of the form `f(x) = mx + b` whose graph is a straight line.
- **Slope (`m`)** — rate of change; rise over run: `m = (y₂ - y₁) / (x₂ - x₁)`
- **Y-intercept (`b`)** — the point where the line crosses the y-axis `(0, b)`.
- **X-intercept** — the point where the line crosses the x-axis; set `y = 0` and solve.

| Slope type | Value | Direction |
|------------|-------|-----------|
| Positive | `m > 0` | Rising left → right |
| Negative | `m < 0` | Falling left → right |
| Zero | `m = 0` | Horizontal line |
| Undefined | vertical | Vertical line |

## ✏️ Worked Examples

### Example 1 — Find slope from two points

**Problem:** Find the slope between `(1, 3)` and `(4, 9)`.

**Solution:** `m = (9 - 3) / (4 - 1) = 6 / 3 = 2`

---

### Example 2 — Write equation in slope-intercept form

**Problem:** Write the equation of the line through `(0, -2)` with slope `3`.

**Solution:** `y = 3x - 2`

## 🧪 Practice Problems

1. Find the slope of the line through `(-1, 5)` and `(3, -3)`.
2. Identify the slope and y-intercept of `y = -4x + 7`.
3. Graph the function `f(x) = 2x - 1` (state at least two points).

<details>
<summary>Answers</summary>

1. `m = (-3 - 5) / (3 - (-1)) = -8 / 4 = -2`
2. Slope: `-4`, y-intercept: `7`
3. Points: `(0, -1)` and `(1, 1)`

</details>
