## 📖 Key Concepts

- **Polynomial** — an expression with one or more terms, each consisting of a coefficient and variable raised to a non-negative integer power.
- **Degree** — the highest exponent of the variable in the polynomial.
- **Monomial** — one term (e.g., `5x²`)
- **Binomial** — two terms (e.g., `x + 3`)
- **Trinomial** — three terms (e.g., `x² + 2x - 1`)

| Degree | Name |
|--------|------|
| 0 | Constant |
| 1 | Linear |
| 2 | Quadratic |
| 3 | Cubic |
| 4 | Quartic |

- **Leading term** — the term with the highest degree.
- **Leading coefficient** — the coefficient of the leading term.

## ✏️ Worked Examples

### Example 1

**Problem:** Identify the degree and leading coefficient of `3x⁴ - 2x² + x - 7`.

**Solution:**
- Degree: `4`
- Leading coefficient: `3`

## 🧪 Practice Problems

1. Classify each as monomial, binomial, or trinomial: `7x³`, `2x - 5`, `x² + 3x + 1`.
2. What is the degree of `4x²y³`?
3. Write a cubic polynomial with leading coefficient `-2`.

<details>
<summary>Answers</summary>

1. `7x³` — monomial; `2x - 5` — binomial; `x² + 3x + 1` — trinomial
2. Degree: `2 + 3 = 5`
3. e.g., `-2x³ + x - 4`

</details>
