## 📖 Key Concepts

- **Adding/Subtracting** — combine like terms.
- **Multiplying** — use the distributive property (or FOIL for two binomials).
- **FOIL** — First, Outer, Inner, Last (for binomial × binomial).

### Special Products

| Pattern | Result |
|---------|--------|
| `(a + b)²` | `a² + 2ab + b²` |
| `(a - b)²` | `a² - 2ab + b²` |
| `(a + b)(a - b)` | `a² - b²` |

## ✏️ Worked Examples

### Example 1 — Addition

**Problem:** Add `(3x² + 2x - 1) + (x² - 4x + 5)`.

**Solution:** `(3+1)x² + (2-4)x + (-1+5) = 4x² - 2x + 4`

---

### Example 2 — FOIL

**Problem:** Expand `(x + 3)(x - 2)`.

**Solution:**  
First: `x²`  Outer: `-2x`  Inner: `3x`  Last: `-6`  
Result: `x² + x - 6`

## 🧪 Practice Problems

1. Subtract: `(5x² - x + 4) - (2x² + 3x - 1)`.
2. Expand: `(2x + 1)(x - 5)`.
3. Use a special product: `(x + 4)²`.

<details>
<summary>Answers</summary>

1. `3x² - 4x + 5`
2. `2x² - 10x + x - 5 = 2x² - 9x - 5`
3. `x² + 8x + 16`
