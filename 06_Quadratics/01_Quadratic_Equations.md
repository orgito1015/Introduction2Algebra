## 📖 Key Concepts

- **Quadratic equation** — an equation of the form `ax² + bx + c = 0` where `a ≠ 0`.
- **Parabola** — the U-shaped graph of a quadratic function `y = ax² + bx + c`.
- **Vertex** — the highest or lowest point of the parabola at `x = -b / (2a)`.
- **Axis of symmetry** — the vertical line `x = -b / (2a)`.
- **Roots / Zeros** — the x-values where the parabola crosses the x-axis (solutions to `ax² + bx + c = 0`).

| `a` value | Parabola opens |
|-----------|---------------|
| `a > 0` | Upward (minimum vertex) |
| `a < 0` | Downward (maximum vertex) |

## ✏️ Worked Examples

### Example 1 — Find the vertex

**Problem:** Find the vertex of `y = x² - 4x + 3`.

**Solution:**  
`x = -(-4) / (2·1) = 2`  
`y = (2)² - 4(2) + 3 = 4 - 8 + 3 = -1`  
Vertex: `(2, -1)`

## 🧪 Practice Problems

1. Identify `a`, `b`, `c` in `3x² - 5x + 2 = 0`.
2. Find the axis of symmetry of `y = -2x² + 8x - 3`.
3. Does `y = x² + 2` cross the x-axis? Explain.

<details>
<summary>Answers</summary>

1. `a = 3`, `b = -5`, `c = 2`
2. `x = -8 / (2 · -2) = 2`
3. No — the minimum value is `2 > 0`, so it never reaches zero.

</details>
