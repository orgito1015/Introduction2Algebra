## 📖 Key Concepts

For `ax² + bx + c = 0`, the solutions are:

```
x = (-b ± √(b² - 4ac)) / (2a)
```

- **Discriminant (`Δ = b² - 4ac`)** — tells you the number and type of solutions:

| Discriminant | Solutions |
|-------------|-----------|
| `Δ > 0` | Two distinct real solutions |
| `Δ = 0` | One real solution (repeated) |
| `Δ < 0` | No real solutions (two complex) |

## ✏️ Worked Examples

### Example 1

**Problem:** Solve `x² - 5x + 6 = 0`.

**Solution:**  
`a = 1, b = -5, c = 6`  
`x = (5 ± √(25 - 24)) / 2 = (5 ± 1) / 2`  
`x = 3` or `x = 2`

---

### Example 2

**Problem:** Solve `2x² + x + 1 = 0`.

**Solution:**  
Discriminant: `1 - 8 = -7 < 0` → **No real solutions**.

## 🧪 Practice Problems

1. Solve: `x² + 6x + 9 = 0`.
2. Solve: `2x² - 4x - 6 = 0`.
3. How many solutions does `3x² - x + 2 = 0` have?

<details>
<summary>Answers</summary>

1. `Δ = 36 - 36 = 0` → `x = -3` (one repeated solution)
2. `a=2, b=-4, c=-6`; `x = (4 ± √(16+48))/4 = (4 ± 8)/4` → `x = 3` or `x = -1`
3. `Δ = 1 - 24 = -23 < 0` → No real solutions

</details>
