## 📖 Key Concepts

**Completing the square** transforms `ax² + bx + c` into `a(x - h)² + k` (vertex form).

**Steps for `x² + bx + c = 0`:**

1. Move the constant to the right: `x² + bx = -c`
2. Add `(b/2)²` to both sides: `x² + bx + (b/2)² = -c + (b/2)²`
3. Factor the left side as a perfect square: `(x + b/2)² = …`
4. Take the square root of both sides.
5. Solve for `x`.

## ✏️ Worked Examples

### Example 1

**Problem:** Solve `x² + 6x + 5 = 0` by completing the square.

**Solution:**  
`x² + 6x = -5`  
Add `(3)² = 9`: `x² + 6x + 9 = 4`  
`(x + 3)² = 4`  
`x + 3 = ±2`  
`x = -1` or `x = -5`

## 🧪 Practice Problems

1. Complete the square: `x² + 4x - 12 = 0`.
2. Write `y = x² - 8x + 7` in vertex form.
3. Solve `x² - 10x = -16` by completing the square.

<details>
<summary>Answers</summary>

1. `(x+2)² = 16 → x = 2` or `x = -6`
2. `(x-4)² - 9` → vertex `(4, -9)`
3. `x² - 10x + 25 = 9 → (x-5)² = 9 → x = 8` or `x = 2`

</details>
