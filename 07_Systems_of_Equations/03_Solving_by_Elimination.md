## 📖 Key Concepts

**Elimination method** (also called the addition method):

1. Multiply one or both equations so that one variable has opposite coefficients.
2. Add the equations together — the target variable cancels.
3. Solve for the remaining variable.
4. Substitute back to find the other variable.

## ✏️ Worked Examples

### Example 1 — Direct elimination

**Problem:** Solve:  
`x + y = 8`  
`x - y = 2`

**Solution:**  
Add the equations: `2x = 10 → x = 5`  
Then `5 + y = 8 → y = 3`  
Solution: `(5, 3)`

---

### Example 2 — Multiply first

**Problem:** Solve:  
`2x + 3y = 7`  
`4x - 3y = 5`

**Solution:**  
Add directly (3y cancels): `6x = 12 → x = 2`  
Then `2(2) + 3y = 7 → y = 1`  
Solution: `(2, 1)`

## 🧪 Practice Problems

1. Solve by elimination: `3x + y = 10` and `3x - y = 2`.
2. Solve: `x + 2y = 6` and `2x - 2y = 0`.
3. Solve: `2x + 3y = 12` and `4x - y = 5`.

<details>
<summary>Answers</summary>

1. Add: `6x = 12 → x = 2, y = 4`
2. Add: `3x = 6 → x = 2, y = 2`
Eliminate `y`: multiply eq2 by 3 to get `12x - 3y = 15`, then add to eq1: `14x = 27 → x = 27/14`. Substitute back: `2(27/14) + 3y = 12 → 3y = 12 - 27/7 = 57/7 → y = 19/7`. Solution: `(27/14, 19/7)`.

</details>
