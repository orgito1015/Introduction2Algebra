## 📖 Key Concepts

| Rule | Formula | Example |
|------|---------|---------|
| **Product** | `aᵐ · aⁿ = aᵐ⁺ⁿ` | `x² · x³ = x⁵` |
| **Quotient** | `aᵐ / aⁿ = aᵐ⁻ⁿ` | `x⁵ / x² = x³` |
| **Power of a power** | `(aᵐ)ⁿ = aᵐⁿ` | `(x²)³ = x⁶` |
| **Power of a product** | `(ab)ⁿ = aⁿbⁿ` | `(2x)³ = 8x³` |
| **Zero exponent** | `a⁰ = 1` (a ≠ 0) | `7⁰ = 1` |
| **Negative exponent** | `a⁻ⁿ = 1/aⁿ` | `x⁻² = 1/x²` |
| **Rational exponent** | `a^(m/n) = ⁿ√(aᵐ)` | `8^(2/3) = 4` |

## ✏️ Worked Examples

### Example 1

**Problem:** Simplify `(3x²y)³`.

**Solution:** `3³ · x⁶ · y³ = 27x⁶y³`

---

### Example 2

**Problem:** Simplify `x⁻³ · x⁵`.

**Solution:** `x⁻³⁺⁵ = x²`

## 🧪 Practice Problems

1. Simplify: `2⁴ · 2⁻²`.
2. Simplify: `(x³)² / x⁴`.
3. Evaluate: `16^(3/4)`.

<details>
<summary>Answers</summary>

1. `2² = 4`
2. `x⁶ / x⁴ = x²`
3. `(⁴√16)³ = 2³ = 8`

</details>
