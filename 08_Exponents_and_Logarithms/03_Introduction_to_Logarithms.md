## 📖 Key Concepts

**Logarithm** is the inverse of exponentiation:

```
logₐ(x) = y   ⟺   aʸ = x
```

- **Common log** — `log(x)` means `log₁₀(x)`.
- **Natural log** — `ln(x)` means `logₑ(x)`.

### Logarithm Properties

| Property | Formula |
|----------|---------|
| **Product** | `logₐ(MN) = logₐM + logₐN` |
| **Quotient** | `logₐ(M/N) = logₐM - logₐN` |
| **Power** | `logₐ(Mᵖ) = p · logₐM` |
| **Identity** | `logₐ(a) = 1` |
| **Zero** | `logₐ(1) = 0` |

## ✏️ Worked Examples

### Example 1 — Convert between forms

**Problem:** Write `log₂(8) = 3` in exponential form.

**Solution:** `2³ = 8`

---

### Example 2 — Expand using properties

**Problem:** Expand `log(x²y)`.

**Solution:** `log(x²y) = log(x²) + log(y) = 2log(x) + log(y)`

---

### Example 3 — Solve a log equation

**Problem:** Solve `log₃(x) = 4`.

**Solution:** `x = 3⁴ = 81`

## 🧪 Practice Problems

1. Evaluate `log₅(125)`.
2. Expand: `ln(x³ / y)`.
3. Solve: `log₂(x + 1) = 5`.

<details>
<summary>Answers</summary>

1. `log₅(125) = log₅(5³) = 3`
2. `3ln(x) - ln(y)`
3. `x + 1 = 2⁵ = 32 → x = 31`
