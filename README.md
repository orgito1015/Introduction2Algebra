## 🗂️ How This Repo Is Organized

Each section lives in its own numbered folder. Inside every folder you'll find:

- **`README.md`** — overview of the section, key concepts, and links to individual notes.
- **Topic files (`.md`)** — detailed notes, definitions, worked examples, and practice problems.

To add a new topic, copy the blank template in [`templates/topic_template.md`](templates/topic_template.md) and fill it in.
